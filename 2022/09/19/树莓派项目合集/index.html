树莓派项目

Chapter2 常用软件和服务

  1. 默认只有vi没有vim，所以要先安装vim编辑器。

    1. 为了解决依赖问题，可以安装aptitude

  2. 树莓派换源

    1. 先用cp备份原来的源文件，然后编辑源文件。（保险，我直接改源文件/etc/apt/sources.list）
    2. 64位系统的话需要增加一个公钥才能够正常使用。
    3. 需要提前查看自己树莓派的系统和架构版本之后才能够选择需要的镜像位置。

  3. 基于Motion的监控

    1. sudo motion开启
    2. sudo killall -TERM motion关闭；
    3. 配置文件sudo vi /etc/motion/motion.conf（回头等到硬盘到了把视频保存在硬盘里）

  4. 文件储存

    1. FTP服务：安装vsftpd服务，启动。配置文件。（sudo或者修改一下权限）

    2. 开启xftp连接树莓派系统。可以直接在Windows上操作文件，很方便。

  5. 文件共享

    1. 安装并ntfs可以让旧的Linux系统识别到NTFS格式的硬盘。（新版的自带）
    2. 安装Samba软件开启文件共享。
      1. Samba本质也是权限管理，在配置文件中添加用户组和权限设置，命令行添加用户之后就可以访问使用。
      2. 4b安装的时候版本会报错，需要先安装正确版本的库。
    3. 开启服务以后就可以正常访问树莓派的公共文件。可以在计算机-添加网络映射中永久性的挂载树莓派。
    4. 暂时无法访问的原因是还没有挂载好硬盘（\media目录下）

  6. Web相关服务

    1. 安装nginx：轻量级的Web服务器，可以反向代理
    2. 内网穿透：外网访问家里的局域网
    3. Aira2
    4. NTP时间服务

  7. 系统备份：全卡备份、压缩备份

  8. MariaDB：数安装据库

  9. dpkg服务是Debian的包管理软件

    1. kill 可以关闭进程。
    2. vim查找使用的是/，使用nN上下翻动。

  10. MarinDB数据库

Chapter3 基于OpenCV的人脸识别

安装配置OpenCV

  1. 扩展TF卡。

  2. 安装开发工具、图像库（视频的相关暂时先不安装了）、GTK和GUI后端。

    1. 过程中同样需要对一些包的版本进行重新安装。如果依赖很多可以使用aptitude代替apt-get。
    2. mv除了可以移动，也可以改名。
    3. OpenCV的安装需要下载之后解压缩

  3. 书上弄了虚拟环境

    1. 不用虚拟环境的方法参照的是博客。[树莓派安装OpenCV完整过程](https://www.cnblogs.com/shanwu369/p/10697216.html#:~:text=树莓派安装OpenCV完整过程. 1. 配置并更新树莓派系统. sudo raspi-config %2F%2F 进入后打开摄像头、SSH sudo,sudo apt-get install build-essential cmake git pkg-config. 3.)

    2. 使用vim需要复制的化需要打入代码

    3. 创建了一个名为cv的虚拟环境并在其中工作。可以看到开头出现了一个(cv)表明环境创建成功。

      1. 使用workon NAME就可以打开虚拟环境

    4. 在虚拟环境中使用Cmake编译

    5. 更改交换机的内存，重新启动一下交换机服务。之后开始编译很长时间。

Chapter5-8 硬件和传感器

GPIO

Python的RPi.GPIO库可以使用树莓派上的GPIO引脚。

相关的通信协议需要下载相应的模块去使用

硬件控制

LED、蜂鸣器、时钟、GPS模块、供电设备、数字键盘、指纹识别、麦克风阵列

电机控制

伺服电机：舵机，就是通过脉冲信号宽度旋转特定角度的电机。

步进电机：单极和双极之分，区别在于后者拥有更大的扭矩，引线更多。

常用传感器

气体传感：二氧化碳、MQ系列（各种燃气的检测）

温湿度：DHT11、气压传感器

电阻式：热敏、光敏

声波传感器：DO、AO、超声波

光：光照强度、红外/紫外、激光、颜色、火焰传感器

力传感器：压力、震动、电子罗盘、角速度和加速度、霍尔传感器。
